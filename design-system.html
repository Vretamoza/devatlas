Design System

poiseUI Style

Un sistema de diseño completo basado en el lenguaje visual de poiseUI. Semántico, componible, con soporte multitema y sin dependencias de JavaScript.

Ver componentes Explorar tokens
Guía

Principios de diseño

⚡
Semántico

Los nombres de clase describen el significado, no la apariencia. btn-primary no bg-purple-600.

🎨
Multitema

Variables CSS para todos los colores. Cambio de tema sin añadir clases adicionales al HTML.

🧩
Componible

Los modificadores de utilidad de Tailwind CSS amplían cada componente sin sobreescribir el sistema.
